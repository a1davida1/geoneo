More tools

Everyday URL checks: use Site check. This page is for HTML reports, citation fixer, Neo Club JSON, fix-tracker POST, pipeline, leads, and advanced audit (market / free-tier fields).

Advanced audit — GET /api/audit

Use for market mode, free package (needs contact fields), or extra parameters. Normal HTTPS sites: Site check.

For package=free, the API also requires contact name, business name, business email, plus URL or industry/location.

Audit HTML report

GET /api/audit-report and download — use a saved auditId from audits.json or the last audit response.

Citation fixer — GET /api/citation-fixer

Neo Club — GET /api/neo-club

Fix-tracker — POST /api/admin/fix-tracker

Operator POST (no membership check). Body must include domain. Optional action":"delete","id":"...".

Pipeline — POST /api/pipeline/:domain

Requires internal Bearer. Body JSON: stage, notes, followUpDate.

Leads — POST /api/leads

Appends to data/leads.json. No auth required.
